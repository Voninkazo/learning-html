HTML rules attributes

What is an attribute?

All HTML elements have attributes that provide an additional information about an element. In most cases they are defined in name="value" pairs and are always declared in the opening tag.
Here is an example:

<tagname attributename="value">
Your content
</tagname>

Syntax rule attributes

When writing an attribute name, there are several rules that we have to follow. So, these followings below will make you aware if all of them.

  • Each attribute values must have double quotes (ex:type="text").
  • Attributes are separated with spaces or new line.
  • Attribute name need to be low case.
  • Some elements types have private attributes id, so the value is unique for that element.
  • Public attributes can go on any elements like class,type,dir and source.
  • Quotes are not allowed inside double qoutes("").

Examples

  1. Do not do this

    <input TYPE=Text>
    This bad html will result invalid markup.

  2. Do this instead

    <input type="text">

NB: Follow those rules when you are writing html, then you will not have any trouble with your code and save your time.
