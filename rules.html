List of all HTML rules

When we are writing code in HTML, there are many rules that we have to follow properly. These are some examples:

  • Define a doctype at the top of an HTML document.
  • You can't have an end tag without a start tag.
  • Every start tag needs to start with a < and end with a >.
  • Every end tag must start with a </ and end with a >.
  • Use lower case instead of uper case for the element name.
  • Commas are not acceptable inside of tags.
  • Have a nice indenting to make your code easy to read, do not start an element iside.
  • Nest your elements properly. If you want more explanation about the nesting rules, please go to this page:
    Learn how to nest elements
  • Pay attention when writing an attribute name, the page below elaborate on the attribute rules:
    See more about attribute rules
  • For high quality code, do test your html in the data validator ,and you will see what is wrong with your code.
  • Do not keep everything that is totaly unecessary, you can do that by questioning everything.
