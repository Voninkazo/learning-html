What is HTML?

This page is helping developers learn more about HTML. HTML, known as Hypertext Markup Language, is a markup language that is used to create documents on the World Wide Web incorporating text, graphics, sound, video, and hyperlinks. It is originally invented in 1992s by Tim Berners-Lee. It describes the structures of a web page by being consisted of series of elements.
