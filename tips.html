Tips for writing HTML

What helps you to write HTML quicker

HTML is a first language that developers learn. As a web developer, speed is useful because it saves our time. These are tips that can help you to improve your speed:

  • Create a template that you can copy code from, that will decrease your typing task and save time as well.
  • Try to remember all of the HTML elements and what they are used for so that you will not struggle to use them.
  • Do not use your mouse very often, use your keybord instead. At first it may be hard for you, but you will be an expert when you get used to it.

What tools shoud we use

There are some tools that we can help you code faster.

  • When you are writing a whole bench of html, use an emmet. This is a tool that will save you lots of typing and will greatly improve your HTML, avoid writing the same thing every time as well. For example when you want to list of links, this is a very quick way to do that:
    Type nav>ul>li*3>a then click Enter
  • Try to use keyboard shortcuts very often. This is much faster than using the mouse.

How do you make sure that your work is high-quality

It is understandable that after writing a lot of code we feel exhausted, but finding time to check our work is very important. So here are some methods that can help you to deal with that:

  • Open your code with browser

Follow these steps:
  1. Save your file with .html extension. If you don't do this, your file will not be read in the browser.
  2. Go to the file location, right clik on that file >open with> and then select the browser of your choice.
  3. And you can keep it open while you are writing code in your .html file and whenever you want to see the changes in your browser, just refresh the page.
