Learn how to nest elements properly

  • Introduction

When you nest an element inside of another element, the nested element becomes a child of the parent element. Many child elements can be placed into a parent element making a sibling relationship among the group of the nested children. If the parent element moves, the children will move automatically with it. So, we need to make sure that the child element is acceptable inside the parent element.

  • Here are some tips that you should be aware of

  1. Don't do this

    <div> <p> Hello world..... </div> </p>

  2. Do this instead

    <div> <p> Hello world..... </p> </div>
